You can view all of the available and configured connections using the <Tooltip tip="Auth0 Dashboard: Auth0's main product to configure your services." cta="View Glossary" href="/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. The configured connections can be used to log in to your applications.

1. Navigate to [Auth0 Dashboard > Authentication](https://manage.auth0.com/#/connections).
2. Select a connection type:

* [Database](https://manage.auth0.com/#/connections/database)
* [Social](https://manage.auth0.com/#/connections/social)
* [Enterprise](https://manage.auth0.com/#/connections/enterprise)
* [Passwordless](https://manage.auth0.com/#/connections/passwordless)
