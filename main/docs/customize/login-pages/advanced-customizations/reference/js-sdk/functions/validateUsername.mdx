---
title: "validateUsername"
description: "Validates a username string against a given username policy. This function checks the username for: - Presence (if no policy is provided) - Minimum and maximum length - Email format restrictions - Phone number format restrictions If no policy is provided, it defaults to checking whether the username is non-empty."
