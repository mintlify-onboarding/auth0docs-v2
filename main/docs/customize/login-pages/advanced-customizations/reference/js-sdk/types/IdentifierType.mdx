---
title: "IdentifierType"
description: "Represents the types of user identifiers that can be used during signup. - `'email'`: The user will provide an email address. - `'phone'`: The user will provide a phone number. - `'username'`: The user will choose a username. This type is typically used in identity selection or signup flows to define which identifiers are enabled and how they behave."
---

## Type Definition

```typescript
type IdentifierType = 'email' | 'phone' | 'username';
```
