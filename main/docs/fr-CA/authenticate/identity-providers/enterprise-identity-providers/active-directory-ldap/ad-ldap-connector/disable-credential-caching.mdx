Vous pouvez désactiver la mise en cache des identifiants au niveau de la connexion des [connexions AD/LDAP](/docs/fr-CA/authenticate/identity-providers/enterprise-identity-providers/active-directory-ldap/ad-ldap-connector) à l’aide d'<Tooltip href="/docs/fr-CA/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip>.

1. Allez dans [Auth0 Dashboard > Authentification > Enterprise](https://manage.auth0.com/#/connections/enterprise), et sélectionnez le type de connexion **Active Directory/LDAP.**
2. Sélectionnez le nom de la connexion que vous souhaitez modifier.
3. Activez le bouton **Désactiver le cache**.
