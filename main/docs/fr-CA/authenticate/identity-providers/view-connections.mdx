Vous pouvez afficher toutes les connexions disponibles et configurées à l’aide du <Tooltip href="/docs/fr-CA/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip>. Les connexions configurées peuvent être utilisées pour vous connecter à vos applications.

1. Allez dans [Auth0 Dashboard > Authentification](https://manage.auth0.com/#/connections).
2. Sélectionnez un type de connexion :

* [Base de données](https://manage.auth0.com/#/connections/database)
* [Social](https://manage.auth0.com/#/connections/social)
* [Entreprise](https://manage.auth0.com/#/connections/enterprise)
* [Sans mot de passe](https://manage.auth0.com/#/connections/passwordless)
