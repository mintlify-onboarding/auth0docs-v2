Pour intégrer Auth0 à une application Web à page unique, vous devez d’abord enregistrer votre application auprès d’Auth0 à l’aide du Dashboard.

1. Naviguez vers [Tableau de bord > Applications > Applications](https://manage.auth0.com/#/applications) et cliquez sur **Créer une application**.
2. Saisissez un nom descriptif pour votre application, sélectionnez **Applications Web à page unique** et cliquez sur **Créer**.

   Une fois la nouvelle application créée, vous serez dirigé vers les **paramètres** de l’**application**, avec les onglets suivants :

## Prochaines étapes

Une fois que vous avez enregistré et configuré votre application, voici quelques étapes courantes à suivre :

* Configurez une connexion et activez-la pour votre application.
* Modifiez le code de votre application pour utiliser votre application enregistrée avec Auth0. Voir nos [Guides de deémarrage rapide Auth0](/docs/fr-CA/quickstarts), où vous trouverez des instructions détaillées et des échantillons pour une variété de technologies. Vous apprendrez également comment implémenter la connexion et la déconnexion, gérer vos sessions utilisateur, récupérer et afficher les informations de profil utilisateur, et plus encore.
* Utilisez les [API Auth0](/docs/fr-CA/api).

  + L'[Authentication API](/docs/fr-CA/api/authentication) gère toutes les fonctions principales liées à l’identité (par exemple, la connexion, la déconnexion et l’obtention du profil utilisateur). La plupart des utilisateurs utilisent cette API à travers nos guides de démarrage rapide, la [bibliothèque Auth0.js](/docs/fr-CA/libraries/auth0js) ou le [widget Lock](/docs/fr-CA/libraries/lock). Cependant, si vous construisez entièrement votre interface utilisateur d’authentification manuellement, vous devrez interagir directement avec cette API.
  + Le [Management API](/docs/fr-CA/api/management/v2) vous permet d’automatiser diverses tâches qui sont également accessibles via le tableau de bord dans Auth0 (par exemple : création d’utilisateurs, configuration des types d’autorisation de l’application).
