Vous pouvez créer un utilisateur en utilisant <Tooltip href="/docs/fr-CA/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip> ou via [Auth0 Management API](https://auth0.com/docs/api/management/v2/users/post-users).

1. Allez à [Dashboard > Gestion des utilisateurs > Utilisateurs](https://manage.auth0.com/#/users) et cliquez sur **Créer un utilisateur**.
2. Saisissez le **Courriel**, le **Mot de passe** et **Confirmation du mot de passe** de votre utilisateur, puis sélectionnez la **Connexion**.

<table class="table"><thead>
<tr>
<th><strong>Attribut</strong></th>
<th><strong>Description</strong></th>
</tr>
</thead>
<tbody>
<tr>
<td>Courriel</td>
<td>L’adresse courriel de l’utilisateur. Le format de l’adresse courriel doit être conforme à la spécification fournie dans <a href="https://datatracker.ietf.org/doc/html/rfc5321#section-4.1.2">RFC 5321 sur IETF</a></td>
</tr>
<tr>
<td>Mot de passe</td>
<td>Le mot de passe de l’utilisateur. Il n’y a pas de limite maximale en ce qui concerne la longueur du mot de passe.</td>
</tr>
<tr>
<td>Répéter le mot de passe</td>
<td>Confirmer le mot de passe.</td>
</tr>
<tr>
<td>Connexion</td>
<td>La connexion à la base de données à utiliser pour authentifier l’utilisateur. Le menu déroulant répertorie toutes les connexions à la base de données configurées dans votre locataire. La connexion que vous utilisez doit être liée à une application, sinon vous recevrez un message d’erreur indiquant que la connexion est désactivée.</td>
</tr>
</tbody>
</table>
3. Cliquez sur **Créer**.
   L’utilisateur est créé et vous serez dirigé vers son profil.

La page **Détails de l’utilisateur** affichera **en attente** lors de la création d’un utilisateur jusqu’à ce qu’il se soit connecté pour la première fois.
