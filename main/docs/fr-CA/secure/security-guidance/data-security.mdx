* [Adresses IP Auth0 pour listes d’autorisations](/docs/fr-CA/secure/security-guidance/data-security/allowlist) – Si un pare-feu protège votre environnement, les connexions personnalisées aux bases de données, les actions, les hooks et les règles peuvent nécessiter l’ajout d’adresses IP Auth0 précises à la liste d’autorisation pour garantir un bon fonctionnement.
* [Ajouter des attributs d’utilisateur à la liste de refus](/docs/fr-CA/secure/security-guidance/data-security/denylist) – Vous pouvez ajouter des attributs d’utilisateur à la liste de refus si certains champs d’utilisateur ne doivent pas se trouver dans les bases de données Auth0 pour des raisons de confidentialité.
* [Stockage de données utilisateur](/docs/fr-CA/secure/security-guidance/data-security/user-data-storage) – Découvrez les différences entre l’utilisation du magasin de données Auth0 et d’une base de données externe pour stocker les données utilisateur.
* [Stockage des jetons](/docs/fr-CA/secure/security-guidance/data-security/token-storage) – Découvrez comment et où stocker les jetons utilisés dans l’authentification basée sur des jetons.
