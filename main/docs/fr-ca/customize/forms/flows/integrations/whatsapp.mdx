## Configurer la connexion à votre coffre-fort

Pour configurer une connexion au coffre-fort pour vos actions WhatsApp, vous aurez besoin d’un **jeton** :

<Frame>![Dashboard > Actions > Forms > Flows > actions > WhatsApp](/docs/images/fr-ca/cdy7uua7fh8z/1Qevi3V1UbkUjb6c51x6R4/44590932bdced6e50afa436835c7cb15/whatsapp-vault-connection.png)</Frame>

Pour savoir comment générer un jeton, lisez l’article [Utilisation de jetons d’autorisation pour WhatsApp](https://developers.facebook.com/blog/post/2022/12/05/auth-tokens/).

Envoie un message.

### Paramètres de saisie

<table class="table">
<thead>
<tr>
<th><b>Paramètre</b></th>
<th><b>Description</b></th>
</tr>
</thead>
<tbody>
<tr>
<td>ID du téléphone expéditeur (obligatoire)</td>
<td>L’ID du téléphone expéditeur se trouve dans les <a href="https://developers.facebook.com/apps">paramètres WhatsApp du Facebook Business Manager</a>. Notez qu’il ne s’agit pas d’un numéro de téléphone.</td>
</tr>
<tr>
<td>Numéro de téléphone du destinataire (obligatoire)</td>
<td>Le numéro de téléphone du destinataire au <a href="https://en.wikipedia.org/wiki/E.164">format E.164</a>.</td>
</tr>
<tr>
<td>Type de message (obligatoire)</td>
<td>
Sélectionnez le type de message dans le menu déroulant. Voici les différents types :
            <ul>
<li>Audio</li>
<li>Contacts</li>
<li>Document</li>
<li>Image</li>
<li>Interactive</li>
<li>Localisation</li>
<li>Autocollant</li>
<li>Modèle</li>
<li>Texte</li>
</ul>
</td>
</tr>
<tr>
<td>Objet de message (obligatoire)</td>
<td>
Le contenu de l’objet de message varie selon le type de message sélectionné. Pour en savoir plus, consultez l’article <a href="https://developers.facebook.com/docs/whatsapp/cloud-api/reference/messages">Messages</a>.
            </td>
</tr>
</tbody>
</table>

### Paramètres de sortie

Vous trouverez les codes d’état attendus dans la [documentation de Facebook](https://developers.facebook.com/docs/whatsapp/cloud-api/support/error-codes).
