Vous pouvez configurer plusieurs locataires pour créer [différents environnements](/docs/fr-ca/get-started/auth0-overview/create-tenants/set-up-multiple-environments) dans Auth0 Dashboard afin de permettre des configurations complexes. Par exemple, vous pouvez avoir deux domaines distincts (un domaine interne et un domaine public), ou vous pourriez souhaiter que les utilisateurs se connectent différemment pour différentes applications. Pour y parvenir, vous pouvez créer plusieurs locataires Auth0 afin de disposer d’ensembles distincts d’applications, de connexions et d’utilisateurs pour les applications et les groupes d’utilisateurs que vous devez prendre en charge.

1. Rendez-vous dans [Auth0 Dashboard](https://manage.auth0.com/#/), sélectionnez votre nom de locataire et sélectionnez **Créer un locataire**.

   <Frame>![Dashboard (Tableau de bord) - Tenant (Locataire) - Menu déroulant - Créer un locataire](/docs/images/fr-ca/cdy7uua7fh8z/53KetqhNIGDs6N5cqJdQtz/dda47c7d1aa9172bf690d6f4e5d17617/2025-02-26_14-49-48.png)</Frame>

2. Saisissez le **Tenant Domain (Domaine locataire)** souhaité, sélectionnez une **Région**, puis cliquez sur **Create (Créer)**.
