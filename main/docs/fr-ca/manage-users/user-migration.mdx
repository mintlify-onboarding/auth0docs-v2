Auth0 prend en charge l’importation des utilisateurs à partir d’applications externes en utilisant des connexions de base de données personnalisées, l’Auth0 <Tooltip href="/docs/fr-ca/glossary?term=management-api" tip="Management API
Un produit permettant aux clients d’effectuer des tâches administratives." cta="Voir le glossaire">Management API</Tooltip> ou l’extension Importation/Exportation des utilisateurs.

## Migrations automatiques

Auth0 prend en charge la migration automatique des utilisateurs depuis une [connexion de base de données personnalisée](/docs/fr-ca/connections/database/custom-db) vers Auth0. Cette pratique est parfois appelée **migration au compte-goutte** ou **migration paresseuse**. Conséquences de cette fonctionnalité pour vos utilisateurs :

* Ils seront déplacés vers Auth0 la première fois qu’ils se connectent après avoir configuré l’intégration.
* Il ne leur sera pas demandé de réinitialiser leur mot de passe à la suite de la migration.

<Card title="La disponibilité varie selon le plan Auth0">

L’implémentation propre à votre connexion et votre plan Auth0 ou accord personnalisé que vous utilisez déterminent si cette fonctionnalité est disponible. Pour en savoir plus, lisez [Tarification](https://auth0.com/pricing).

</Card>

Lorsqu’un utilisateur s’authentifie au moyen d’une connexion de base de données marquée pour importation vers Auth0, le processus suivant se produit :

* Auth0 authentifie les utilisateurs migrés par rapport à la base de données Auth0.
* Si l’utilisateur n’a pas été migré, Auth0 exécute votre script de connexion personnalisé et, une fois la connexion réussie, ajoute l’utilisateur à la base de données Auth0.
* Lors des connexions ultérieures, les informations d’identification de l’utilisateur sont extraites d’Auth0, **PAS** de votre base de données personnalisées.
* Les nouveaux utilisateurs sont automatiquement ajoutés à la base de données Auth0.

## Importations d’utilisateurs en masse avec Management API

Si vous avez déjà une base de données utilisateurs, vous pouvez utiliser notre [`/post_users_imports`](/docs/fr-ca/api/management/v2#!/Jobs/post_users_imports) point de terminaison de Management API pour renseigner une connexion de base de données avec ces informations.

## Migrer les utilisateurs à l’aide de l’extension Importation/Exportation des utilisateurs

L’extension Importation/Exportation des utilisateurs vous permet de faire ce qui suit :

* Importer en masse les utilisateurs de votre base de données existante vers Auth0.
* Rechercher et exporter une partie (ou la totalité) des utilisateurs de votre base de données Auth0.

Vous devez être Administrateur du Dashboard pour utiliser cette extension.

Vous pouvez importer et exporter des données utilisateur à l’aide de l’extension d’importation/exportation qui figure à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard. Sélectionnez l’extension **Importation/Exportation des utilisateurs** et effectuez l’installation. Pour en savoir plus, consultez [l’extension Importation/Exportation des utilisateurs](/docs/fr-ca/manage-users/user-migration/user-import-export-extension).
