<Card title="Avant de commencer">

* Abonnez-vous à un plan Entreprise avec module complémentaire Authentification multifacteur (MFA) adaptative. Consultez [Tarification Auth0](https://auth0.com/pricing/) pour plus de détails.
* Configurez et activez une connexion à une base de données ou à Active Directory.
* Configurez et activez au moins un facteur MFA.

</Card>

Utilisez l’authentification multifacteur (<Tooltip href="/docs/fr-ca/glossary?term=multifactor-authentication" tip="Authentification multifacteur (MFA)
Processus d’authentification de l’utilisateur qui utilise un facteur en plus du nom d’utilisateur et du mot de passe, tel qu’un code par SMS." cta="Voir le glossaire">MFA</Tooltip>) adaptative pour activer la MFA lorsque Auth0 détermine qu’une tentative de connexion est à risque et pour enregistrer les évaluations de risque pour toutes les transactions de connexions de vos bibliothèques de locataires.

## Activer l’authentification multifacteur (MFA) adaptative

Vous pouvez activer l’authentification multifacteur (MFA) adaptative dans le Auth0 Dashaboard ou avec <Tooltip href="/docs/fr-ca/glossary?term=management-api" tip="Management API
Un produit permettant aux clients d’effectuer des tâches administratives." cta="Voir le glossaire">Management API</Tooltip>.

<Tabs><Tab title="Dashboard">

1. Allez à [**Dashboard > Security (Sécurité) > Multi-factor Auth (MFA)**](https://manage.auth0.com/#/security/mfa).

<Frame>![Auth0 Dashboard Security Multi-factor Auth Adaptive MFA Policy](/docs/images/fr-ca/cdy7uua7fh8z/4IlQi0LXOPJdYjOuo09xtE/92a6214a5a14ae1d9abbd358fd59cf7e/2025-02-28_13-38-54.png)</Frame>

2. Dans la section **Factors (Facteurs)**, activez et configurez au moins un facteur MFA. Pour en savoir plus, lisez [Facteurs d’authentification multifacteur (MFA)](/docs/fr-ca/secure/multi-factor-authentication/multi-factor-authentication-factors).

3. Dans la section **Define policies (Définir les politiques)**, repérez **Require Multi-factor Auth (Exiger la MFA)**, puis sélectionnez **Use Adaptive MFA (Utiliser l’authentification multifacteur (MFA) adaptative)**. L’évaluation des risques sera automatiquement activée et enregistrée dans les journaux de vos locataires.

4. Cliquez sur **Enregistrer**.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">

Si vous utilisez le facteur `email` de l’[authentification Identifier First](/docs/fr-ca/authenticate/login/auth0-universal-login/identifier-first), vous devez mettre à jour les attributs du courriel dans [Dashboard > Connexions à la base de données > Méthodes d’authentification](https://manage.auth0.com/#/connections/database). Dans l’onglet Configuration du courriel, assurez-vous que l’attribut email est actif. Ensuite, définissez **Autoriser l’inscription sur obligatoire** et activez **Exiger** l’adresse électronique dans le profil utilisateur.

<Frame>![Auth0 Dashboard (Tableau de bord Auth0) > Authentication (Authentification) > Database Connections (Connexions à la base de données) > Authentication Methods (Méthodes d’authentification)](/docs/images/fr-ca/cdy7uua7fh8z/ql7yYX1GnJaiPQnjQQBTs/f06f659ab8f094ce26003ccb859a57e0/Email_Config_-_French.png)</Frame>

</Callout>

</Tab><Tab title="Management API">

1. Obtenez un [jeton d’accès à Management API](/docs/fr-ca/secure/tokens/access-tokens/management-api-access-tokens/get-management-api-access-tokens-for-production) avec la permission `update:mfa_policies`.

2. Appelez le point de terminaison [Définir les politiques d’authentification multifacteur (MFA)](https://auth0.com/docs/api/management/v2/guardian/put-policies) de Management API avec la charge utile appropriée.

</Tab></Tabs>

## Activer l’évaluation de risque Authentification multifacteur (MFA) adaptative

Si vous n’êtes pas prêt à activer l’authentification multifacteur (MFA) adaptative mais que vous souhaitez commencer à l’entraîner pour analyser les comportements de connexion, vous pouvez activer l’évaluation de risque de l’authentification multifacteur (MFA) adaptative indépendamment.

1. Allez dans [Dashboard > Sécurité > Multi-factor Auth](https://manage.auth0.com/#/security/mfa).
2. Localisez la section **Définir les politiques**.
3. Dans **Évaluateurs du risque pour la MFA**, sélectionnez **Activer l’évaluation de risque de l’authentification multifacteur (MFA) adaptative**.
4. Sélectionnez **Enregistrer**.

## Personnaliser l’authentification multifacteur (MFA) adaptative

Vous pouvez personnaliser le comportement de l’authentification multifacteur (MFA) adaptative pour fournir la meilleure expérience pour vos utilisateurs tout en assurant leur sécurité. Pour en savoir plus, lisez [Personnaliser l’authentification multifacteur (MFA) adaptative](/docs/fr-ca/secure/multi-factor-authentication/adaptive-mfa/customize-adaptive-mfa).

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">

Les actions qui déclenchent la MFA sont prioritaires sur le comportement par défaut de l’authentification multifacteur (MFA) adaptative.

</Callout>

## Limites

Les informations d’évaluation dans les journaux de locataires ne sont disponibles que pour les flux interactifs Auth0 ne prend pas en charge l’enregistrement des informations d’évaluation pour les flux d’autorisation de mot de passe de propriétaire de ressources (<Tooltip href="/docs/fr-ca/glossary?term=resource-owner" tip="Propriétaire de ressources
Entité (telle qu’un utilisateur ou une application) capable d’accorder l’accès à une ressource protégée." cta="Voir le glossaire">Resource Owner</Tooltip> Password Grant, ROPG) sans l’authentification multifacteur (MFA) adaptative. Pour plus d’informations sur les limitations du flux d’authentification, lisez [Authentification multifacteur (MFA) adaptative](/docs/fr-ca/secure/multi-factor-authentication/adaptive-mfa).
