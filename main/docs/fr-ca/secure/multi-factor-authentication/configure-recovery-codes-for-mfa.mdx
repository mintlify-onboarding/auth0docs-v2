Un code de récupération correspond à un code unique, généré par Auth0, permettant à un utilisateur de retrouver l’accès à son compte. Ainsi, si un utilisateur ne peut pas accéder à l’appareil ou au compte utilisé pour l’inscription à l’authentification multifacteur (<Tooltip href="/docs/fr-ca/glossary?term=multifactor-authentication" tip="Authentification multifacteur (MFA)
Processus d’authentification de l’utilisateur qui utilise un facteur en plus du nom d’utilisateur et du mot de passe, tel qu’un code par SMS." cta="Voir le glossaire">MFA</Tooltip>), il peut utiliser un code de récupération pour s’authentifier.

## Fonctionnement

Lorsque vous utilisez la connexion universelle avec les codes de récupération activés :

1. Un utilisateur commence l’inscription MFA.
2. Auth0 génère un code de récupération.
3. Durant le processus d’inscription MFA, l’utilisateur voit l’invite du code de récupération.
4. L’utilisateur enregistre le code de récupération et termine le processus d’inscription.
5. Ce dernier peut désormais terminer l’authentification MFA avec le code de récupération enregistré s’il perd l’accès à son appareil ou au compte qu’il a inscrit à l’authentification MFA.

<Frame>![Exemple d’invite de code de récupération lors de la demande d’authentification multifacteur](/docs/images/fr-ca/cdy7uua7fh8z/6LkXKthRal1r4vIPi3137l/4652ed114b1128c050027b3f409f5baa/2025-02-26_14-32-27.png)</Frame>

Lorsque vous utilisez la connexion universelle avec les codes de récupération désactivés :

* Les utilisateurs ne verront pas l’invite du code de récupération lors de l’inscription MFA.
* Les utilisateurs ne peuvent pas s’authentifier à l’aide d’un code de récupération.

## Activer les codes de récupération

Les codes de récupération sont désactivés par défaut. Vous pouvez activer les codes de récupération en vous dirigeant vers [Dashboard > Sécurité > Authentification multifacteur](https://manage.auth0.com/#/security/mfa).
