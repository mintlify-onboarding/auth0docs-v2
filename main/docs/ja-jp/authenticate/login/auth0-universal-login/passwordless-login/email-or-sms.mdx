Auth0では、ユニバーサルログインを構成して、ユーザーがマジックリンクやワンタイムパスワード（OTP）をメールで受け取って認証したり、SMS経由でOTPを受け取って認証したりできます。

SMSまたはメールを使用したパスワードレス認証を有効にするには、アプリケーションの[パスワードレス接続](/docs/ja-jp/authenticate/passwordless)を定義する必要があります。

接続を作成したら、これを使用するようにユニバーサルログインを構成する必要があります。これには3つのオプションがあります。

## ユニバーサルログイン

この方法では、パスワードレス接続をネイティブにサポートするユニバーサルログインエクスペリエンスを使用します。詳細については、「[ユニバーサルログインを使用したパスワードレス](/docs/ja-jp/authenticate/passwordless/passwordless-with-universal-login)」をお読みください。

## ユニバーサルログイン＋Lock（パスワードレス）

この方法は、 **Lock（パスワードレス）** テンプレートとカスタムログインページを使用して、[Lock for Web SDK](/docs/ja-jp/libraries/lock)でユーザーを認証します。

1. [［Dashboard］>［Branding（ブランディング）］>［Universal Login（ユニバーサルログイン）］](https://manage.auth0.com/#/login_settings)に移動して、 **［Login（ログイン）］** タブをクリックします。
2. **［Custom Login Page（カスタムログインページ）］** のトグルを有効にして、 **［Lock（パスワードレス）］** テンプレートを選択します。HTMLテンプレートがパスワードレスのカスタマイズオプションを使用したLockウィジェットのコードで更新されます。
3. テンプレートをカスタマイズし、 **［Save Changes（変更の保存）］** をクリックします。

HTMLとCSSを使用してログインフォームをカスタマイズし、Dashboard内で変更をプレビューすることができます。 **Lock（パスワードレス）** テンプレートのカスタマイズ方法については、[Lock for Web SDK](/docs/ja-jp/libraries/lock)の「パスワードレス認証」セクションをお読みください。

## ユニバーサルログイン＋カスタムUI＋Auth0.js

この方法では、 **カスタムログインフォーム** テンプレートとカスタムログインページを使用して、[Auth0.js SDK](/docs/ja-jp/libraries/auth0js)でユーザーを認証します。詳細については、「[ユニバーサルログインを使用したパスワードレス認証](/docs/ja-jp/authenticate/passwordless/implement-login/universal-login)」をお読みください。
