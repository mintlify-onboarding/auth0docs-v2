## Dashboardを使用してフックシークレットを表示します

1. [［Auth0 Dashboard］](https://manage.auth0.com/#/)の[［Hooks（フック）］](https://manage.auth0.com/#/hooks)ページに移動し、編集するフックの横にある鉛筆アイコンをクリックします。
2. フックエディターで、レンチアイコンをクリックし、 **［Secrets（シークレット）］** をクリックします。
3. **［Add Secret（シークレットを追加）］** をクリックします。
4. シークレットのわかりやすい名前と値を入力し、 **［Save（保存）］** をクリックします。

## Management APIを使用してフックシークレットを作成する

[Add Hook Secrets エンドポイント](/docs/ja-jp/api/management/v2/#!/Hooks/post_secrets)に `POST`呼び出しを行います。`MGMT_API_ACCESS_TOKEN`、`HOOK_ID`、`HOOK_SECRET_KEY`、および`HOOK_SECRET_VALUE`のプレースホルダー値は、それぞれManagement APIアクセストークン、フックID、およびHook key-valueペアに置き換えてください。

<AuthCodeGroup>
```bash cURL
curl --request POST \
  --url 'https://{yourDomain}/api/v2/hooks/HOOK_ID/secrets' \
  --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN' \
  --header 'cache-control: no-cache' \
  --header 'content-type: application/json' \
  --data '{ [ "HOOK_SECRET_KEY", "HOOK_SECRET_VALUE" ], [ "HOOK_SECRET_KEY", "HOOK_SECRET_VALUE" ] }'
```

package main

import (
	"fmt"
	"strings"
	"net/http"
	"io/ioutil"
)

func main() {

	url := "https://{yourDomain}/api/v2/hooks/HOOK_ID/secrets"

	payload := strings.NewReader("{ [ "HOOK_SECRET_KEY", "HOOK_SECRET_VALUE" ], [ "HOOK_SECRET_KEY", "HOOK_SECRET_VALUE" ] }")

	req, _ := http.NewRequest("POST", url, payload)

	req.Header.Add("content-type", "application/json")
	req.Header.Add("authorization", "Bearer MGMT_API_ACCESS_TOKEN")
	req.Header.Add("cache-control", "no-cache")

	res, _ := http.DefaultClient.Do(req)

	defer res.Body.Close()
	body, _ := ioutil.ReadAll(res.Body)

	fmt.Println(res)
	fmt.Println(string(body))

}
