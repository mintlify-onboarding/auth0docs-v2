## 注意や警告の使用は正しく

注意と警告は、果たす役割が異なります。注意では、知っておくと便利な一般情報を紹介し、警告には、従わないとエラーにつながってしまう情報を記載します。

警告に値する情報があるときは、それに関連する作業をユーザーが行う箇所に記載します。警告は、ユーザーが探さなくても目に入らなければなりません。
