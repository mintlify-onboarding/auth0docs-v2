ユニバーサルログインは、ログイン、<Tooltip data-tooltip-id="react-containers-DefinitionTooltip-0" href="/docs/ja-jp/glossary?term=multifactor-authentication" tip="多要素認証（MFA）: ユーザー名とパスワードに加えて、SMS経由のコードなどの要素を使用するユーザー認証プロセス。" cta="用語集の表示">MFA</Tooltip>要素の登録、パスワードのリセットなど、アカウントに関連したさまざまな操作を実行できる一連のページで構成されています。Auth0はそれらのページに複数の言語でテキストを提供しています。

場合によっては、このテキストをアプリケーションの文体や特定のニーズに合わせて変更する必要があるかもしれません。

[ユニバーサルログイン](/docs/ja-jp/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience)ページに表示されるテキストは、<Tooltip data-tooltip-id="react-containers-DefinitionTooltip-0" href="/docs/ja-jp/glossary?term=auth0-dashboard" tip="Auth0 Dashboard: サービスを構成するためのAuth0の主製品。" cta="用語集の表示">Auth0 Dashboard</Tooltip>または<Tooltip data-tooltip-id="react-containers-DefinitionTooltip-0" href="/docs/ja-jp/glossary?term=management-api" tip="Management API: 顧客が管理タスクを実行できるようにするための製品。" cta="用語集の表示">Management API</Tooltip>を使用して、すべての対応言語でカスタマイズすることができます。どちらの方法でも以下の定義、値、変数が適用されます。

## プロンプトと画面を構成する

利用可能な[プロンプトと画面](#prompt-values)は、Auth0 Dashboardまたは[Management API](/docs/ja-jp/api/management/v2)を使用してカスタマイズすることができます。

<Tabs><Tab title="Dashboardの使用">

Dashboardを使用して、ユニバーサルログインのテキストプロンプトをカスタマイズできます。

1. Dashboardの[**［Branding（ブランディング）］ > ［Universal Login（ユニバーサルログイン）］ > ［Advanced Options（高度なオプション）］**](https://manage.auth0.com/#/custom_text)に進み、**［Custom Text（テキストのカスタム）］** を選択します。
2. 編集したい **［Prompt（プロンプト）］** と **［Screen（画面）］** を選択し、希望の変更を行います。右側に公開プレビューが表示されます。
3. 変更を運用環境に反映させたい際は、**［Save Changes（変更の保存）］** を選択します。

</Tab><Tab title="Management APIの使用">

Management APIを使用して、ユニバーサルログインテキストプロンプトをカスタマイズできます。

1. `read:prompts`と`update:prompts`スコープを持つ[Management APIのアクセストークン](/docs/ja-jp/secure/tokens/access-tokens/management-api-access-tokens/get-management-api-access-tokens-for-production)を取得します。
2. Management APIの[プロンプトのカスタムテキストの取得](https://auth0.com/docs/api/management/v2#!/Prompts/get_custom_text_by_language)エンドポイントを呼び出し、指定プロンプトと言語に対する既存の構成データを取得します。
3. 参考のために [サポートパラメーター](#supported-parameters)と[プロンプト値](#prompt-values)を使用して、希望の構成データを修正してください。
4. Management APIの更新された構成データを持つ[指定プロンプトにカスタムテキスト設定](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language)エンドポイントを呼び出します。

   <Warning>

   Management APIの[指定プロンプトに対するカスタムテキストの設定](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language)エンドポイントは、既存の全構成データを指定プロンプトの`body`パラメーターに入力された構成データに置き換えます。

   複数の`screen`値を構成したい場合には、それらをすべてManagement APIに対する1つの呼び出しに含める必要があります。

   </Warning>

#### 例

Management APIの[指定プロンプトにカスタムテキスト設定](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language)エンドポイントを呼び出し、 `Log in to ACME's website`の`login`プロンプトの`login`画面に使用する`description`テキストコンポーネントを設定することができます。次のようなものです：

```bash cURL
curl --request PUT \
  --url 'https://{yourDomain}/api/v2/prompts/login/custom-text/en' \
  --header 'authorization: Bearer {yourMgmtApiAccessToken}' \
  --header 'content-type: application/json' \
  --data '{ "login": { "description": "Login to ACME'''s Website" } }'
```
```csharp C#
var client = new RestClient("https://{yourDomain}/api/v2/prompts/login/custom-text/en");
var request = new RestRequest(Method.PUT);
request.AddHeader("content-type", "application/json");
request.AddHeader("authorization", "Bearer {yourMgmtApiAccessToken}");
request.AddParameter("application/json", "{ "login": { "description": "Login to ACME's Website" } }", ParameterType.RequestBody);
IRestResponse response = client.Execute(request);
```
```go Go
package main

import (
	"fmt"
	"strings"
	"net/http"
	"io/ioutil"
)

func main() {

	url := "https://{yourDomain}/api/v2/prompts/login/custom-text/en"

	payload := strings.NewReader("{ "login": { "description": "Login to ACME's Website" } }")

	req, _ := http.NewRequest("PUT", url, payload)

	req.Header.Add("content-type", "application/json")
	req.Header.Add("authorization", "Bearer {yourMgmtApiAccessToken}")

	res, _ := http.DefaultClient.Do(req)

	defer res.Body.Close()
	body, _ := ioutil.ReadAll(res.Body)

	fmt.Println(res)
	fmt.Println(string(body))

}
```

#### プロンプトのカスタムテキストの削除

プロンプトのすべてのカスタムテキストを削除したい場合、Management APIの[指定プロンプトにカスタムテキストの設定](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language)エンドポイントを呼び出し、`body`パラメーターとして空のオブジェクト（`{}`）を指定してください。
