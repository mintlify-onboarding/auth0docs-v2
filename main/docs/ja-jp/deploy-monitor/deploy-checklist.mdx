Auth0では、以下のデプロイメントチェックリストをご提供しています。記載されている項目がすべて該当するわけではないため、各自の実装に合わせてチェックリストを適宜調整してご利用ください。

[**アーキテクチャ** ](https://auth0.com/docs/get-started/architecture-scenarios)

[**ユーザーのプロビジョニング** ](https://auth0.com/docs/manage-users)

[ユーザーの認証](https://auth0.com/docs/get-started/identity-fundamentals/authentication-and-authorization)

[ユーザープロファイルの管理](https://auth0.com/docs/manage-users/user-accounts/user-profiles)

[ユーザーのログアウト](https://auth0.com/docs/authenticate/login/logout)

[デプロイメントの自動化](https://marketplace.auth0.com/integrations/github-actions)

デプロイメントの段階では、システムをステージング環境や運用環境にデプロイし、ユーザーが操作や使用を実際に開始します。

最終的なリリースでは、システムにあるコンポーネントがすべて運用環境にデプロイされます。
