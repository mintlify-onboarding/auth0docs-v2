Dashboardを使用すると、アプリケーションの署名アルゴリズムを変更することができます。

1. [［Dashboard］>［Applications（アプリケーション）］>［Applications（アプリケーション）］](https://manage.auth0.com/#/applications)に移動し、アプリケーションの名前を選択して表示します。

   <Frame>![Dashboardのアプリケーションリスト](/docs/images/ja-jp/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/6a547af05f8729ea00700a851e13c62b/App_List_-_JP.png)</Frame>

2. **［Settings（設定）］** タブの下までスクロールし、**［Show Advanced Settings（詳細設定を表示）］** を選択します。
3. **［OAuth］** タブに移動します。

   <Frame>![Dashboard アプリケーション アプリケーションの設定タブ 高度な設定 OAuthタブ](/docs/images/ja-jp/cdy7uua7fh8z/5bbq6VrI9tUMht0PwMimre/04454945e118ce22b1287ce0b0dffb45/OAuth_-_Japanese.png)</Frame>

4. **［JsonWebToken Signature Algorithm（JsonWebToken署名アルゴリズム）］** を見つけて、アプリケーションに適した署名アルゴリズムを選択します。
5. **［Save Changes（変更を保存）］** をクリックします。
