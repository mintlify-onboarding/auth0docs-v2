ログインセッションを使用すると、<Tooltip data-tooltip-id="react-containers-DefinitionTooltip-0" href="/docs/ja-jp/glossary?term=auth0-dashboard" tip="Auth0 Dashboard: サービスを構成するためのAuth0の主製品。" cta="用語集の表示">Auth0 Dashboard</Tooltip>管理者は、複数のデバイスとブラウザからの異なるダッシュボードアプリケーションのアクティブセッションを追跡できます。Teams Dashboard、サポートセンター、およびAuth0 Dashboardにアクセスするユーザーが作成したセッションを確認し、取り消すことができます。

Auth0 Dashboardの[プロファイルページ](https://manage.auth0.com/#/profile/sessions)で、すべてのアクティブセッションのリストを見ることができます。

<Frame>![Auth0 Dashboard > Profile](/docs/images/ja-jp/cdy7uua7fh8z/3hBNRo2Bc4w6oKpF0mMt4o/00454b132aad16642225266c19c7d9ba/User_Login_Sessions.png)</Frame>

セッションをログアウトまたは取り消すには、対象のセッションを見つけて **［Revoke（取り消す）］** を選択します。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">

セッションを取り消しても、拡張機能に属するセッションは自動的に削除されません。ただし、拡張機能に属するセッションは存続期間が短く、ログインセッションが取り消されたり期限切れになったりしても更新されません。

</Callout>
