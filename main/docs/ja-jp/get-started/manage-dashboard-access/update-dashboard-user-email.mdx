テナント管理者は、新しいメールアドレスを持つ新しいユーザーを追加し、古いメールアドレスに関連付けられているユーザーを削除することで、既存の<Tooltip data-tooltip-id="react-containers-DefinitionTooltip-0" href="/docs/ja-jp/glossary?term=auth0-dashboard" tip="Auth0 Dashboard: サービスを構成するためのAuth0の主製品。" cta="用語集の表示">Auth0 Dashboard</Tooltip>メンバーに関連付けられているメールアドレスを更新できます。

1. [［Dashboard］>［Settings（設定）］>［Tenant Members（テナントメンバー）］](https://manage.auth0.com/#/tenants/admins)に移動します。
2. **［Add Member（メンバーの追加）］** をクリックします。
3. 新しいメールアドレスを入力し、アクセス許可を選択します。
4. **［Invite（招待）］** をクリックして、ユーザーにメール招待を送信します。
5. ユーザーが招待を承諾したら、古いメールアドレスに関連付けられているユーザーを削除します。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">

エンドユーザーがDashboardやAPIを使って自分のメールアドレスを更新することはできません。アカウントに関連付けられているメールアドレスを更新する必要がある場合は、管理者に連絡してください。

</Callout>
