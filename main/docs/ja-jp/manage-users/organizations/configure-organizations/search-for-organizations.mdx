[組織](/docs/ja-jp/manage-users/organizations/organizations-overview)を扱う際に、名前や表示名で組織を検索しなければならないことがあるかもしれません。これには、<Tooltip data-tooltip-id="react-containers-DefinitionTooltip-0" href="/docs/ja-jp/glossary?term=auth0-dashboard" tip="Auth0 Dashboard: サービスを構成するためのAuth0の主製品。" cta="用語集の表示">Auth0 Dashboard</Tooltip>を使うことができます。

<Frame>![Auth0 Dashboard - 組織の検索オプション - 名前と表示名](/docs/images/ja-jp/cdy7uua7fh8z/5CeeKLjPGxjTqy682lbnyj/7efb298d6963439ae337f7ade28d3c00/2025-01-27_17-26-50.png)</Frame>

入力したテキストが検索されたフィールドの中に見つかった場合には、結果が返されます。たとえば、「Big Holdings Co」という名前の組織があり、検索文字列が「`hold`」や「`ings`」であった場合には、Big Holdingsの組織が返されます。

組織の検索では、最大50件の結果が返されます。望み通りの結果が得られない場合には、より具体的な検索文字列を指定してください。

組織の検索は結果整合性のある機能です。最近作成や変更を行った組織は検索結果に含まれない可能性がありますが、いずれは反映されます。
