無料サブスクリプションプランをご利用の方を含む、すべてのお客様は、Auth0の[Responsible Disclosure Program](https://auth0.com/responsible-disclosure-policy)でセキュリティに関する懸念を報告することができます。法的情報については、[法的詳細](https://auth0.com/legal)をお読みください。
