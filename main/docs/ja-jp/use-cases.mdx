<Warning>

フォームは現在、早期アクセスで使用できます。Auth0のリリース段階については、「[製品のリリース段階](/docs/ja-jp/troubleshoot/product-lifecycle/product-release-stages)」を参照してください。

</Warning>

アクション用のフォームを使用することで、特定のユースケースに合わせてAuth0のサービスをカスタマイズできます。
以下のフォームのユースケースを作成および設定できます。

* **プログレッシブプロファイリング** を使用して、ユーザー情報を段階的に収集するためのルールセットを定義する。
* **カスタムサインアップステップ** を使用して、サインアップまたはログインフローをステップやフィールドで拡張し、カスタマイズする。
* [更新ポリシーフォームの設定](https://sus.auth0.com/docs/contentful/7ZaefQOyKukQKq8p31lTS)を使用して、ユーザーがアプリケーションにログインするために同意を要求する。
