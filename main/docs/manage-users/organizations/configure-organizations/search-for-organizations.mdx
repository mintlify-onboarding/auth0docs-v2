From the Organizations section of your [Auth0 Dashboard](https://manage.auth0.com/#/organizations), you can search for [organizations](/docs/manage-users/organizations/organizations-overview) by name or display name.

<Frame>![Auth0 Dashboard - Organizations - Search Bar ](/images/cdy7uua7fh8z/5CeeKLjPGxjTqy682lbnyj/3a41b87a369e65ca624b1822c99dbc57/2025-01-27_17-27-28.png)</Frame>

The dropdown menu beside the search bar allows you to search by Name or Display Name. Entering three or fewer characters into the search bar will return exact matches only. If you enter four or more characters, fuzzy matches containing the string provided will be returned. For example, if you have an organization named "Big Holdings, Co.", and you enter the search terms `hold` or `ings`, the Big Holdings, Co. organization will be returned.

Searches return a maximum of 50 results. If the desired result isnâ€™t found, try using a more specific search term.

<Warning>

The organization search employs eventual consistency. Recently created or modified organizations may not immediately display in search results; however, the results will eventually accurately reflect the entities within your tenant.

</Warning>
